Functional


F_1 - FunctionPort Ports are not allowed on non-leaf functions (meaning that a functional exchange has not been allocated down to a leaf function).

FCH_1 - FunctionalChain FunctionalChain should involve only leaf functions.

FC_1 - Component Components and actors should be connected by at least one CE
FC_2 - ComponentExchange CE should have at least one allocated FE
FC_3 - ComponentExchange FE between components/actors should be allocated to CE
Note: not tested when connected to an actor whose name contains:'User' (case-sensitive).
FC_4 - ComponentPort Port with CE should have provided/required Interface
FC_5 - FunctionalExchange FunctionalExchange allocated to a CE should have one or more Exchanged Items
FC_6 - FunctionalExchange FunctionalExchanges on same input port must have same name
FC_7 - FunctionalExchange FunctionalExchanges on same input port must have same EI
FC_8 - FunctionalExchange FunctionalExchanges on same output port must have same name
FC_9 - FunctionalExchange FunctionalExchanges on same output port must have same EI
FC_10 - SystemFunction Function name shall be unique
FC_11 - Actor Actors name shall be unique
FC_12 - Interface@SA The interface EI shall be identical at SA to the CE EI allocated.
FC_13 - ComponentExchange@SA The CE EI shall be the union of the EI of the FE allocated to the CE
FC_14 - AbstractFunction FunctionalExchanges incoming with same name must use same port
FC_15 - AbstractFunction FunctionalExchanges outgoing with same name must use same port
