Design > Consistency


DCON_01 - Pattern Conformance
The rule checks that model elements which are bound to a pattern conform to the pattern.


DCON_02 - RPL to REC conformance
This rule checks if a RPL is up to date compared to its origin REC.


DCON_03 - Invalid pattern instance
The rule checks if pattern instances still point to existing Capella elements.


DCON_04 - RPL/REC_RPL without REC
This rule checks if a RPL or REC_RPL has no REC.


DCON_05 - REC might not have any RPL instance
This rule checks if a REC might not be utilised in a RPL.


DCON_06 - REC_RPL might not have any RPL instance
This rule checks if a REC_RPL might not be utilised in a RPL.


DCON_07 - RPL shall not reference an element of its REC
This rule checks if a RPL references an element which is also referenced by its REC


DCON_08 - REC/RPL detect no source no target
REC_RPL no target or no source
