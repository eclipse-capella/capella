How to import a Capella Project Example?

Sample projects can be imported using File / Import, then Capella Project Example:



So far the only sample model provided is the IFE sample model.
Select the sample model to import then select Finish.



The sample model is imported and the Activity Explorer is automatically open.

    The zip file for the sample model is provided in the Capella installation folder under the sub-folder samples.

How to add an existing Capella project as an example?

  1. The sample project must be a zip containing the project folder
  2. The sample project shall be placed in sample folder like CAPELLA_INSTALLATION_PATH/samples/model.zip or located under a sub folder like CAPELLA_INSTALLATION_PATH/samples/model/model.zip
  3. The import sample wizard dialog will propose a list of available examples including the newly added ones.
