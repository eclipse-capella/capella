How to rename a Capella Project?

Either the session is closed or the session is open, you can rename the project.

The following sequence must be respected:

  1. On the project, type F2 (or contextual menu “Rename”): this action will rename the folder of the project and modify the “.project” file inside this folder. It will rename also the Project root element of the model and the System Engineering element. If the project contains images referenced in the diagram, the paths also to those images in the aird will be updated.
  2. On the “ .capella” file, type F2 (or contextual menu “Rename”): this action will rename the “.capella” file and update all references from the “.aird” files to the “.capella” file.
  3. On the “ .aird” file, type F2 (or contextual menu “Rename”): this action will rename the “.aird” file and also the associated “.afm” file and update all references in the “.aird” files to the “.afm” file.
  4. On the “ .melodyconnector” file, type F2 (or contextual menu “Rename”): this action will rename the “.melodyconnector” file.


The same process can be applied on Library projects.


If you rename a Library project, all the references to this library will be lost. Libraries renaming must be done very carefully.
