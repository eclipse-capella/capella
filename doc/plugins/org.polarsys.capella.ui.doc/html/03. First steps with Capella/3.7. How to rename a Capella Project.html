How to rename a Capella Project?

The first step is to close the session, if it’s opened.

Then, the following sequence must be respected:

  1. On the project, type F2 (or contextual menu “Rename”): this action will rename the folder of the project and modify the “.project” file inside this folder.
  2. On the “.melodymodeller” file, type F2 (or contextual menu “Rename”): this action will rename the “.melodymodeller” file, the Project root element of the model and update all references from the “.aird” files to the “.melodymodeller” file.
  3. On the “.aird” file, type F2 (or contextual menu “Rename”): this action will rename the “.aird” file and also the associated “.afm” file (so the step 5 becomes optional) and update all references in the “.aird” files to the “.afm” file.
  4. On the “.melodyconnector” file, type F2 (or contextual menu “Rename”): this action will rename the “.melodyconnector” file.
  5. On the “.afm” file, type F2 (or contextual menu “Rename”): this action will rename the “.afm” file and update all references in the “.aird” files to the “.afm” file.


The same process can be applied on Library projects.


If you rename a Library project, all the references to this library will be lost. Libraries renaming must be done very carefully.


If you rename a Capella project or a Library project, all your Orchestra links might be broken. Projects renaming in Orchestra environment must be done very carefully.
