API changes from Capella 6.x to 7.x

Please also refer to Sirius Release Notes.

Major changes

Migration to Eclipse 2023-03

Capella 7.x is built with Eclipse 2023-03.

Version major increased

Metamodels have been updated to 7.0.1.

Diagram capabilities

Paste options were renamed and a new option Paste Style was added.

A new option was added that allow to paste quickly image from clipboard on nodes in diagram.

It was added a customizable color menu easy for selecting the last used colors or the selected color on a diagram element and reuse them on other elements.

Diagram Toolbar buttons are accessible in all cases (e.g. small or multiple diagram Editor).

Linked ports are now properly laid out when D&D in diagram or automatically inserted at Diagram refresh.

Semantic Browser

Semantic Browser has been improved to show non oriented Physical Link ends.

Label decorations in Semantic Browser are now present and consistent with the Project Explorer.

Migration

Capella 7 supports the migration of Capella models from Capella 1.4 - i.e. from Capella 1.4.x, 5.x, 6.x.
