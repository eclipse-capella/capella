LITERAL (NUMERIC, STRING, BOOLEAN, ENUMERATION)

DEFINITION

Literals allow defining fixed Data Values.

Examples of Numeric Literals for a Numeric Type of Kind FLOAT:

  • 25
  • +6.34
  • 0.5
  • 2,50E-02
  • -1

Examples of String Literals:

  • 'Alex Ovechkin'
  • 'NONE'
  • 'January 2002'

Examples of Boolean Literals:

  • YES, TRUE, ON
  • NO, FALSE, OFF
